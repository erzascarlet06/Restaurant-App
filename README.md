## Technology details :

1. React Native
2. AWS Amplify
3. Amplify Auth
4. Amplify Datastore
5. DynamoDB

## Requirements :
1. react-native: v18.2.0
2. npm : v9.5.0 
3. aws-amplify: v5.0.22,

## Steps to Setup

**1. Clone the application**

```bash
git clone <git URL>
```

**2. Build and run the app**

```bash
npm run dev
```
